Abstract

The aim of the project is to provide a secure and comprehensive web platform for NHS to post challenges (with its accompanying data) which are solvable by AI. The developers can work on the challenges on the platform itself and thereafter submit their solutions which will be ranked according to its accuracy.

Our solution is a Django-based web project which are then further split into smaller Django applications e.g. challenges, solutions, users etc which handles different areas of the project.

As of the prototype phase, the front-end has been completely set up. For the backend, the database modes, the user authentication system and JupyterHub has been completed.

Key features of your project

  • Full-fledged user authentication and personalised profile
  • Differing functionality and interface for the two different user types (clinician & developer)
  • Ability to create challenges and upload data for clinicians
  • Ranking of solutions by accuracy
  • Provision of on-site coding environment based off Jupyter
